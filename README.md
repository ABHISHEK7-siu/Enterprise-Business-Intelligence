## 🌟 Featured Project: Omnichannel E-Commerce Analytics Suite (Olist Brazil)
*Tools Deployed:* Microsoft Power BI, Tableau, Advanced Excel, Power Query

### 📊 Business Scenario
Managed a highly complex, 9-table relational database tracking 100k+ real-world transactions from Olist Store in Brazil. Preprocessed data using Power Query to translate localized Portuguese product schemas into English, eliminate structural data gaps, and handle regional geospatial mapping.

### 🚀 Key Technical Architectures
* *Relational Data Modeling:* Architected a production-ready *Star Schema* linking fact transaction data with dimension profiles (Customers, Sellers, Products, Geolocation).
* *Advanced DAX Engineering:* Formulated dynamic Time-Intelligence measures (YTD Profit, Next Year Sales, QTD Sales) to deliver period-over-period financial performance comparisons.
* *UI/UX Navigation Canvas:* Implemented a central interactive home page navigation menu with modular button routing to drill deep into Sales, Profits, and Business Indicators.
